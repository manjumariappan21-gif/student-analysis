Student Performance Analysis Project

Overview:

This project analyzes the academic performance of 250 students to identify patterns, trends, and factors affecting their scores. It uses data analysis and machine learning techniques to generate insights and predictions.

Dataset:
Total Students: 250

Features:

Name:
Department (CSE, IT, ECE)
Year (1st, 2nd, 3rd)

Gender:

Marks (Maths, Science, English)
